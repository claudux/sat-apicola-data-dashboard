# Contexto de negocio

Este proyecto fue desarrollado para un profesional del área agrícola que necesitaba una forma clara y confiable de visualizar el diagnóstico inicial de productores apícolas de la Región de Los Ríos.

La información de origen provino de entrevistas realizadas a productores de miel. A partir de esos datos, se llevó a cabo un proceso de limpieza, estructuración y análisis para transformar los registros levantados en información útil para la toma de decisiones.

El dashboard resultante permitió visualizar variables relevantes como georreferenciación, distribución por género, kilos producidos, costos estimados e implementación de laboratorios, entre otros indicadores clave del diagnóstico.

La solución final fue de alto valor para el profesional solicitante, ya que permitió diferenciarse por presentar conclusiones precisas en un formato claro, visual y respaldado por datos confiables.
