# Proceso de limpieza y preparación de datos

La base de datos utilizada en este proyecto provino de entrevistas realizadas a productores apícolas. Como parte del trabajo de preparación, fue necesario consolidar, revisar y estructurar la información para habilitar su análisis y visualización posterior.

## Principales desafíos
- consolidación de información levantada en terreno
- revisión de consistencia entre registros
- normalización de categorías
- preparación de variables territoriales y demográficas
- organización de indicadores productivos y económicos

## Acciones realizadas
- revisión general de la estructura de la base
- limpieza de campos y validación de registros
- normalización de categorías de análisis
- preparación de variables de georreferenciación
- estructuración de métricas de producción, costos e ingresos
- segmentación por comuna, género, edad y tipo de tenencia

## Resultado
El proceso permitió transformar una base de diagnóstico en una fuente de información confiable y útil para construir un dashboard claro, visual y orientado a gestión.
